# simple kernel
> simple kernel with makefile

### prepare
> $ sudo apt-get install build-essential linux-headers-$(uname -r)

### use
> $ make <br>
> $ sudo insmod main.ko <br>
> $ modinfo main.ko <br>
> $ tail /var/log/kern.log <br>
